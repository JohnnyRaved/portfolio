Local Weather

Local Weather demonstrates manipulation of the Google Maps and places APIs. The user may enter a location with autocompletion assistance, or accept the default locaton as determined by geolocation.

Technologies: HTML 5, CSS, JavaScript 5, Bootstrap 3, jQuery 2.2, jQuery UI 1.1

Wikipedia Viewer

Wiki Viewer takes a search term provided by the user and returns a list of up to fifty topics. The user may hover over a topic to view a tooltip containing a summary of the related article topic.

Technologies: HTML 5, Bootstrap 3, jQuery UI 1.1

Twitch Streamers

Twitch viewer enables the user to view a set of Twitch users (streamers), along with their online status. Clicking on the description area of an online streamer takes the user to the streamer's channel. Demonstrates a custom object to marshall asynchronous calls within an array.

Technologies: HTML 5, Bootstrap 3, jQuery UI 1.1
